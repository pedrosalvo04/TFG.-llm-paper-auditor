Item 9 (Code of Ethics):
  - Evaluate the SUBSTANCE of ethical consideration, not a literal string match.
  - A dedicated 'Ethics Statement', 'Broader Impacts' section, or an explicit declaration of adherence to ANY recognised scientific code of ethics (NeurIPS, ACL, ICLR, IEEE, etc.) is sufficient for a "Yes".
  - ONLY answer "No" if there are CLEAR ETHICAL RED FLAGS (e.g., harmful applications, dual-use risks, human subject experiments) without any ethics statement, IRB discussion, or ethical justification whatsoever.
  - DO NOT penalise a paper merely for citing the wrong conference code name. Focus on whether the authors demonstrate genuine ethical awareness.
